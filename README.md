# Website Paroki

This is the official repository for the Website Paroki project. It is built using [Vue 3](https://vuejs.org/) and [Vite](https://vitejs.dev/), styled with [Tailwind CSS](https://tailwindcss.com/) and [Bootstrap](https://getbootstrap.com/), and uses [Prismic](https://prismic.io/) as the Headless CMS.

## Prerequisites

Before you begin, ensure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (v16.0.0 or higher recommended)
- npm (usually comes with Node.js)

## Getting Started

Follow these steps to get a local copy of the project up and running.

### 1. Clone the Repository

Open your terminal and run the following command to clone the repository:

```bash
git clone <repository-url>
cd website-paroki
```

> **Note**: Replace `<repository-url>` with the actual URL of this Git repository.

### 2. Install Dependencies

Install the project dependencies using npm:

```bash
npm install
```

### 3. Run the Development Server

Start the local development server:

```bash
npm run dev
```

After running this command, you should see output indicating that the server is running (usually at `http://localhost:5173`). Open this URL in your browser to view the project.

## Scripts

- **`npm run dev`**: Starts the development server with hot-reload.
- **`npm run build`**: Builds the project for production.
- **`npm run preview`**: Locally preview the production build.

## Tech Stack

- **Framework**: Vue 3
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, Bootstrap
- **CMS**: Prismic
